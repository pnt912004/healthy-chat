# AGENT Protocol

General rules and communication protocols for AI Agents (Gemini & Claude).

## Communication Rules
- Use Markdown for all documentation.
- Maintain a clear and professional tone.
- Reference `memory.md` before making significant architectural decisions.
- Update `task.md` after completing a significant unit of work.

## AI-to-AI Handover
- When switching between agents, provide a brief summary of current progress and next steps.
